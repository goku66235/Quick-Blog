# 🚀 Full Stack Blog Application (MERN)

A production-ready full-stack blog platform built using the MERN stack, featuring user and admin functionalities with secure authentication and real-time API integration.

---

## 🔗 Live Demo
👉 https://quick-blog-a5fb.onrender.com

---

## ✨ Features

### 👤 User Side
- Browse and read blogs
- Responsive UI for all devices
- Dynamic content loading

### 🔐 Admin Dashboard
- Create, edit, and delete blog posts
- Publish / Unpublish blogs
- Manage content efficiently

### 🔒 Authentication
- Secure login system
- Role-based access control (Admin/User)
- Protected routes

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Axios
- Vite

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- REST APIs
- Git & GitHub
- Render (Deployment)
